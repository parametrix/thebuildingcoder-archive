Autodesk Lab Projects Spark and Storm

Two new interesting BIM projects are now available on Autodesk Labs:

Project Spark

Project Spark is a technology preview of a simplified 3D building information modelling (BIM) solution. It enables building professionals to create designs efficiently with real-world building objects, produce more reliable documentation faster, and share files with consultants using Revit or AutoCAD-based products. It does not support conceptual design, analysis, rendering, advanced collaboration, and, unfortunately for us developers, API.

Project Spark is intended to help users who are new to BIM get up and running efficiently. In addition to simplified functionality it includes additional user assistance such as essential skills videos and a Getting Started Guide which has step by step instructions for creating a model with accompanying video demonstrations.

Project Storm

Project Storm for Revit Structure is available on Autodesk Labs!

It is a technology preview that provides cloud-based structural analysis to structural engineers as a part of the BIM process. It enables structural engineers and designers to perform static analysis in the cloud directly within Revit Structure. Results can be visualized and explored within Revit Structure. Disruptions to workflows are minimized by performing analysis as you continue designing.
