Chinese Revit 2012 API Webcast

My colleague Joe Ye of the Beijing office is presenting a Chinese version of the recent Revit 2012 API Webcast on Thursday, June 2nd.

As always, to register, you can visit the Autodesk Developer Network (ADN) DevTech API training schedule site, filter for 'Revit API', and click on the register link. This webcast is free of cost and open to all developers, regardless of ADN membership.

Here is the Chinese agenda:

  • Revit 2012 API SDK
  • 2011 Revit 2012
  • Revit 2012 APIs
      API
    • Revit Structure API
    • Break
    • Revit MEP API
    • SDK samples

    The English agenda was given in the previous post.

    Project Vasari 2.0

    An updated version 2.0 of Project Vasari is now available, compatible with Revit 2012. It is of interest to Revit API developers, since it includes part of the API.

    BIM Family Toolkit for Revit 2012

    The BIM Family Toolkit has also been updated and is now available for Inventor 2012 and Revit 2012.
