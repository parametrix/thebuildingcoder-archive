Room Renumbering Plugin of the Month

Our latest Plugin of the Month has been released on Autodesk Labs: RoomRenumbering for Revit.

For the first time ever, this month's plugin is a Revit add-in, a room renumbering application by Adam Nagy.

It includes source code and full documentation and is intended to be simple enough for any end user interested in customisation to understand and be able to extend for her personal use, so be sure to check it out for yourself!
