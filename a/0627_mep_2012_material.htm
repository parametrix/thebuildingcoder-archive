MEP Sample Code for Revit 2012

One of my sessions at Autodesk University 2011 is CP4453 Everything in Place with Revit MEP Programming. It will cover both the new Revit MEP 2012 API features and the Revit MEP API in general.

In preparation for that, as well as preparing new samples, e.g. on placeholder elements, I am also updating the existing material, especially the ADN Revit MEP sample code.

The old Revit 2011 material from AU 2010 is available online, for instance from the list of all sessions in the Revit API track at AU 2010 and for Revit MEP from my Revit MEP 2011 API class materials. They are also available directly from the CP316-3U All Systems Go in Autodesk Revit MEP Programming page in the AU site itself.

So, after all that retrospective introspection, here is rme_src_2011-08-09.zip containing the flat migrated sample material for Revit MEP 2012. The descriptions of the features implemented provided by last year's handout material still apply.
