Revit 2012 WikiHelp Overview

Here is a product related news item, although obviously of interest to developers as well.

Revit product help has been available on the web for a while now, which provides a number of benefits.

Some online Revit API help (link unavailable) is also available, although the full Revit SDK documentation is not there yet.

Now the product help system has taken the next step by moving to a wiki-based system which provides support for a full community learning experience.
